# 📊 Walmart Sales Performance Dashboard
Personal Project | 2026

## Overview
Built Excel and Power BI dashboard analyzing sales trends 
across 45 Walmart stores from 2010 to 2012. Identified 
top-performing stores and seasonal patterns to support 
business decision-making.

## Tools Used
- Power BI Desktop
- Microsoft Excel
- Python (Pandas)

## Dataset
- 6,435 weekly sales records
- 45 stores across 3 years (2010–2012)

## Key Insights
- Store 20 is top performer — $301M total sales
- 2011 was best year — $2.45B total sales
- July is highest revenue month — $650M
- Holiday weeks generate 7.8% higher avg sales
- January is lowest month — post-holiday slowdown

## Files
- Walmart_Sales_Dashboard.pbix — Power BI dashboard
- Walmart_Sales_Dashboard_FINAL.xlsx — Excel dashboard
